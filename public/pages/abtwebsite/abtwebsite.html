about the website

✧✦ suggestions ✧

I did my best to make sure everyone could have a pleasing browsing experience on the site, but I am no professional coder, this is purely a hobby of mine. If you encounter any bugs, please let me know, I'll do my best to fix them. If you have any suggestions on how I can improve my already existing code, I would be thrilled to know, just contact me anywhere. If you have any suggestions about pages I could add or cool features you'd like to see added here, also let me know!

✧✦ licensing ✧

This website is open source, all the code and assets can be found here.
You can use and modify the code however you wish, no attribution necessary, but I certainly wouldn't mind it if you linked my page somewhere on your site! If you have any questions about how anything works here, feel free to reach out to me and ask.

✧✦ development stuff ✧

  • Made for and hosted on Neocities. A mirror of the site is available on Github at amendokat.github.io.
  • Developed on Visual Studio Code, with Live Server and Code Spell Checker & Firefox Developer Edition on the Windows 10 OS.
  • Tested on Firefox Developer Edition and Microsoft Edge for desktop & Firefox for mobile.
  • All files are hosted on Github and deployed to Neocities via deploy-to-neocities.
  • robots.txt provided by ai.robots.txt. ai.txt provided by Blocking AI Crawlers.

✧✦ accessibility and responsiveness ✧

There are no major accessibility or responsiveness issues from what I've encountered. You should be able to navigate the site smoothly regardless of the device you're using.

Implemented features

  • Mobile layout.
  • All gifs are blurred and have opacity lowered for users with reduced motion preferences turned on. obs: unfortunately this can erroneously include certain static images, as the style that targets it specifically looks for the .gif extension.
  • Images feature alt text or alternatively a caption describing it's contents.
  • Images outside of landing page feature lazy loading.
